# Support

Nexus is a volunteer-maintained source beta.

## Before opening an issue

1. Confirm macOS 14 or newer and Node.js 22 or newer.
2. Run `npm ci`, `npm run check`, and the smallest relevant test.
3. Search existing issues.
4. Remove API keys, prompts, model responses, file names, file contents, recordings, diagnostics, and personal data.

Use the bug template for reproducible defects and the proposal template for feature requests. Model availability, provider billing, account access, rate limits, and provider-side retention are controlled by OpenAI or Anthropic and should be handled through those providers.

## Security

Do not post exploitable details publicly. Follow [SECURITY.md](SECURITY.md) and GitHub private vulnerability reporting.

## Distribution

The supported public artifact is currently the source repository. Do not download Nexus installers from third-party mirrors. A release is supported only when it is published from this repository with checksums and verified signing/notarization status.
