# Changelog

Nexus follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and semantic versioning while the public API stabilizes.

## [Unreleased]

- Signed and notarized macOS distribution is blocked until Developer ID and Apple notary credentials are configured in GitHub Actions.

## [0.1.0] - 2026-07-10

### Added

- Local-first Electron workspace with Solo and two-provider Council conversations.
- macOS Keychain provider credentials, local SQLite history, files, voice, research, MCP, generated skills, and explicit privileged-action approvals.
- Sender-validated IPC, restrictive Chromium permissions and CSP, read-only command broker, constrained MCP environments, and bounded research jobs.
- Local retention, export, deletion, optional personalization, and reviewed feedback-package controls.
- Public Next.js website, product documentation, security and privacy pages, structured data, social images, sitemap, robots, manifest, and `llms.txt`.
- Open-source governance, security automation, SBOM generation, release checksums, and guarded signing/notarization workflow.

### Distribution

- Source beta only. No signed or notarized installer was published with this entry.
